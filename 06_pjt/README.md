    - 회원의 로그인, 비로그인 상태에 따라 다른 링크 출력하기
    - Reset 버튼으로 사용자의 모든 입력을 초기 값으로 재설정 하기
    - 팔로우, 팔로잉, 좋아요 수를 출력
